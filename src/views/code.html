基于 Luminol 系服务端的 Minecraft 公益群组服

LuminolCraft 是基于 Luminol 系服务端的 Minecraft 公益群组服，综合生存与纯净生存两开花给予玩家多种选择，支持多端互通且无正版门槛，兼具稳定运行与备份保障，融入特色玩法，与玩家共创纯粹又富体验感的生存空间。

Features

服务器特点

我们致力于为玩家提供最优质的游戏体验

响应迅速

服务器管理团队响应迅速，时刻关注玩家需求与反馈，及时处理各类问题，确保游戏环境稳定有序。无论是规则答疑、矛盾调解还是技术故障，都能快速给出解决方案，让玩家无后顾之忧。

多端畅玩

支持 Java 版和基岩版多端互通，并且无需正版账号即可游玩，降低了玩家的加入门槛，让更多人能参与其中，享受跨平台的游戏乐趣。

安全可靠

服务器 24×7 小时稳定运行，拥有较高的 TPS；每 4 小时会自动备份存档，保障玩家的创造成果安全；同时管理活跃，能快速解决问题，维持和谐的环境。

Game Modes

服务器类型

两种不同的游戏模式，满足不同玩家的需求

纯净模式

纯净生存

纯净生存，保持纯净的原版生存。仅保留 /tpa、/home、/rtp 等基础命令且设冷却，避免过度插件干扰原生态体验；不限制红石，不更改原版特性，没有圈地等插件，管理活跃保证玩家的创造。主世界不开启死亡不掉落，强化生存挑战的真实感，而下界与末地则开启该功能，平衡探索风险；让玩家在原汁原味的收集、建造、冒险中，感受恰到好处的挑战与自由。

综合模式

综合生存

融合 RPG 生存、技能系统与粘液科技，开启无限可能的冒险世界。手持神器，探索副本，体验 RPG 战斗的畅快淋漓。自动化生产，感受粘液科技的高效魔力。技能系统融入日常生存，开启你的独特"修仙"之旅。领地系统如同卫士，为你的奇思妙想保驾护航。死亡不掉落，冒险更安心。一个世界，玩法多样！立即加入，与伙伴一起探索无限可能！
